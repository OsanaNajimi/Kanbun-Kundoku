漢文訓読文を書き下し文に変換するプログラムです。

### 入力フォーマット

1行目に漢文を記述し、2行目に対応する返り点を記述します。  
返り点がない文字にはアンダーバー（`_`）を使用してください。  
2文字をつなげる場合は、2文字目にハイフン（`h`）を記載してください。

例：
```
聞大王有意督過之
2 _ _ r1 r 2 h 1
```

###### 返り点のエンコーディング対応表：

| 返り点  | エンコーディング |
| ---- | -------- |
| なし   | __       |
| レ    | r        |
| 一    | 1        |
| 二    | 2        |
| 三    | 3        |
| 上    | u        |
| 中    | m        |
| 下    | d        |
| ハイフン | h        |
